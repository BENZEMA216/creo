# Brand: Seek Within

## Identity
- **Slogan**: "You might not need that many clothes / 你或许不需要那么多衣服"
- **Philosophy**: 长期主义着装哲学 — 克制美学、高利用率、少即是多。"让穿衣回归纯粹，你只需专注生活本身"
- **Target audience**: 25-35岁都市女性，追求高智感、复古气质、舒适与高级兼得
- **Price positioning**: 中高端国货 — ¥179-¥389
- **Key differentiator**: 进口高端面料（Supima棉、奥地利兰精莫代尔/天丝、莱赛尔羊毛、蚕丝羊毛）+ 克制极简设计 + 一衣多穿结构

### Typography & Layout (from PDF catalogs)
- **Heading**: 中文黑体（衬线感）+ 英文衬线体（类似 Didot/Bodoni 风格），大写，宽字距
- **Body**: 中文宋体/细黑体，英文无衬线，清晰简洁
- **品牌名排版**: "SEEK WITHIN" 大写衬线，字距极宽，配细横线分隔
- **季节标记**: "26SS" 独立右上角，粗体
- **Layout pattern**: 左侧大图（模特全身/细节）+ 右侧文字信息区，三栏式产品图排列
- **Whitespace**: 极大留白，呼吸感强，文字区域紧凑但不拥挤
- **信息层级**: 产品名 → 颜色/款号 → 设计理念 → 面料 → 版型 → 细节 → 穿搭 → 尺码表

### Photography Style
- **Lighting**: 柔和自然光，均匀无强烈阴影，干净通透
- **Backgrounds**: 纯色浅灰/白色无缝背景纸，极简无干扰
- **Model styling**: 短发模特（Boyish风），极简妆容，无夸张配饰，偶有丝巾/眼镜点缀
- **Composition**: 正面全身 + 背面细节 + 局部特写三组合；产品图有平铺和穿着对比
- **Post-processing**: 低对比度、微暖调、去除杂色，保持面料质感清晰
- **道具**: 极简 — 偶尔出现金属椅（极简家具）、白色百叶裙等质感道具
- **产品摄影**: 白T有包装盒+品牌卡+吊牌的开箱式拍摄，强调仪式感

### Tone of Voice
- **Writing style**: 诗意、克制、内省。短句为主，逗号断句，留白感
- **Key phrases**: "存在感的消失"、"让穿衣回归纯粹"、"都市穿行，自在如风"、"呼吸之间"、"日常的雕塑"、"高利用率长期主义"、"克制美学"
- **Copywriting patterns**:
  - 产品名偏文艺："风行裤"、"显瘦神裤"、"闲散漫步"
  - 面料描写拟人化/通感："面料，即是风的形态"、"奶油般丝滑"、"云感软糯"
  - 设计理念引用年代："70年代复古灵感"、"90年代智慧美学"
  - 系列命名英文化：Daily、Noble、Ballet、Office Siren
- **禁忌**: 不用促销语言（"清仓"、"秒杀"），不用网红口语（"绝绝子"、"yyds"），保持高级调性

## Design Patterns Summary (for quick reference)

### 做设计时遵循
- 大面积留白 + 低饱和配色
- 中英文混排：中文为主叙述，英文用于系列名/品牌名
- 产品图三视角：正面全身 + 背面/侧面 + 局部特写
- 信息架构：视觉（大图）在前，文字说明在后
- 字体气质：衬线感标题 + 细黑体正文，不用圆体/可爱字体
- 色彩：以黑白灰为基底，每季加入 2-3 个低饱和彩色

### 写文案时遵循
- 短句、留白、诗意。避免感叹号和网络用语
- 面料是核心卖点，用通感手法描写（触觉→视觉→情感）
- 强调"不需要那么多"的反消费主义叙事
- 系列命名用英文，产品描述用中文
